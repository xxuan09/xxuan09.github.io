# 💭 心情记录 - 云端日记

一个温暖的心情和日常记录页面，用浮出的小云来记录你的每一个想念时刻。

## ✨ 功能特性

### 左侧面板 - 心情选择
- 🎭 6种心情选择：开心、难过、疲惫、兴奋、平静、焦虑
- 😸 对应的猫表情包自动生成
- ☁️ 每次选择心情都会浮出一朵小云

### 右侧面板 - 计分板
- 📊 "等你来更新" 计分板
- 💕 点击按钮记录"有人在想你"
- 📈 实时显示计数
- 💭 随机浮出温暖的提醒

### 温暖的提醒系统
- 📢 根据不同心情生成对应的提醒语
- 🌈 每次点击都有随机的温暖信息
- 💾 数据自动保存到本地存储

## 🚀 如何使用

1. **打开页面**：访问 https://xxuan09.github.io
2. **选择心情**：点击左侧的心情选项
3. **记录想念**：点击右侧的"有人在想你"按钮
4. **看云朵浮起**：每次操作都会有小云浮出
5. **查看提醒**：右侧会显示温暖的想念提醒

## 🎨 设计特点

- 💜 紫色梦幻渐变背景
- ☁️ 光滑的浮动云朵动画
- 📱 完全响应式设计，支持手机和平板
- ✨ 流畅的交互动画和过渡效果

## 💾 数据持久化

- 使用浏览器 localStorage 保存计数器
- 数据在浏览器中持久存储，关闭页面后仍然保留
- 支持每天重置计数器（可选）

## 🔧 自定义

你可以修改以下内容来个性化这个页面：

- **更改心情**：在 `script.js` 中的 `moodData` 对象中修改
- **更换颜色**：修改 `style.css` 中的梯度颜色
- **修改提醒语**：在 `moodData` 的 `messages` 数组中编辑
